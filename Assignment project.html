What's the Difference?

HTML

Hypertext Markup Language

Create the structure
  • Controls the layout of the content
  • Provides structure for the web page design
  • The fundamental building block for any web page

CSS

Cascading Style Sheet

Stylize the website
  • Applies style to the web page elements
  • Targets various screen sizes to make the web page responsive
  • Primarily handles the "look and feel" of a web page

Java Script

Increase interactivity
  • Adds interactivity to a webpage
  • Handles complex functions and features
  • Programmatic code which handles functionality
